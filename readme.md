# Devcamper API

> Backend API for DevCamper application, bootcamp directory website.

## Usage
Update "config/config.env" according to your enviroment

## Install dependencies

```
npm install
```

## Run App 

```
# Run in dev. mode
npm run dev

#Run in production mode
npm start
```
## Deployed API:
```
 https://backend-api-v1.herokuapp.com/
```
